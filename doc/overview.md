# Pocket Ledger 项目总览

## 产品定位

随手记账微信小程序 + 配套 Web 网页，支持多种方式录入账单，调用 GLM 模型智能解析，提供多维度查账功能。

---

## 功能清单

### 微信小程序

| 模块 | 功能 |
|------|------|
| 登录 | 微信一键登录，首次绑定手机号 |
| 记账 | 拍照上传、选择PDF、录音输入、手动数字输入 |
| 记账类型 | 收入 / 支出 |
| 查账 | 按月、季度、年度查看账单列表 |

### Web 网页

| 模块 | 功能 |
|------|------|
| 登录 | 手机号 + 短信验证码 |
| 查账 | 图表展示、按时间/类别筛选 |
| 账单管理 | 编辑、删除、分类 |

---

## 基建架构

```
用户端
├── 微信小程序（原生 WXML/JS）
└── Web 前端（Vue3，部署到腾讯云静态托管）

后端
└── 腾讯云云托管（Node.js）

存储
├── 云数据库（MySQL）
├── 云存储（图片 / PDF / 音频文件）
└── 云开发内置缓存（短信验证码）

外部服务
├── 智谱AI GLM API（账单内容解析 + 语音转写）
└── 短信服务（阿里云/腾讯云，手机号验证码）
```

---

## 域名与证书

- 使用已有域名，新增子域名（如 `ledger.yourdomain.com`）
- SSL 证书使用腾讯云/阿里云免费版，90天自动续签
- 小程序后端接口域名需在微信公众平台配置合法域名白名单

## 用户体系

- 小程序端：`wx.login()` 获取 openid，首次进入引导绑定手机号
- Web端：手机号 + 验证码登录，后端通过手机号查到对应 openid，签发同一用户 JWT token
- 两端数据完全共享同一用户账号
